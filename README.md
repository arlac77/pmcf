# pmcf

## Poor mans configuration management

Generates system packages from config blueprint.

supported linux derivates:

*   arch linux
*   debian
*   rpm based systems

generates config packages for:

*   systemd
    *   networkd
    *   timesyncd
    *   resolverd
*   kea
*   bind
*   mosquitto
*   influxdb
*   iwd
*   chrony
